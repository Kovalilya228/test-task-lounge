# Inventory App (Vue 3 + Pinia)

Небольшое приложение для работы с инвентарём: добавление предметов, изменение количества и подсчёт общего веса.

## Стек

- Vue 3 (Composition API)
- Pinia
- TypeScript
- Vite
- обычный CSS

## Что умеет

- добавлять и удалять предметы из инвентаря
- увеличивать / уменьшать количество
- автоматически считать общий вес и количество
- список с прокруткой
- фиксированный summary внизу
- адаптив под мобильные устройства (карточки вместо списка)

## Как устроено

### Store (Pinia)
Вся логика хранится в одном store:

- `items` — список всех предметов
- `inventory` — состояние инвентаря
- `totalCount` — общее количество предметов
- `totalWeight` — общий вес

### UI

- `InventoryList` — список предметов
- `InventoryItem` — один элемент инвентаря
- `InventorySummary` — итоговые значения

## Оптимизация (что сделал)

- доступ к данным через Map (быстрее поиск)
- убрал лишние вычисления в компонентах
- список перерисовывается только при изменении нужных данных
- разделил UI и store логику
- оптимизация рендера списка предметов с помощью tanstack Virtual
- ленивая загрузка изображений + fallback в случае ошибки

## Адаптив

- на десктопе — обычный список
- на мобильных — карточки
- summary всегда внизу и не скроллится
